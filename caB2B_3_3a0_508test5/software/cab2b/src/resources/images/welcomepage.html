Welcome to caBench-to-Bedside


caBench-to-Bedside (caB2B) is an application that facilitates translational research by integrating disparate molecular, biospecimen, pathology, imaging, and clinical data sets and by co-analyzing and co-visualizing this information.

In the first iteration, the goals are to:

  • Search for biospecimens and associated microarray expression profiling data
  • Annotate microarray data sets using information available from biomedical databases
  • Co-analyze microarray data sets and clinicopathology information
  • Examine the results using a rich set of visualization windows

caB2B allows bench scientists, translational researchers, and clinicians to:

Step 1: Search Data
Search for and add data sets (e.g. microarray data and associated pathology information) to an experiment
Step 2: Analyze and Visualize Data
Analyze (e.g. cluster) and visualize (e.g. in the context of pathways) data sets within an experiment.
