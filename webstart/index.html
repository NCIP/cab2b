Welcome to caBench-to-Bedside (caB2B)

The NCI caBIG™ project creates a common, extensible platform that integrates diverse data types and supports interoperable analytic tools. caBIG™ is developing applications to facilitate individual steps involved in micro-array analysis. caGrid, an infrastructure for caBIG, is used to perform investigations involving integration of data and analytical services from diverse research communities. caB2B leverages these tools in a user-friendly graphical user interface (GUI). caB2B can be used by the physician scientist to perform operations such as the following:

  • Query any caGrid data service to obtain data.
    caB2B enables investigators to query tissue banks at multiple cancer centers, enabling studies to focus on very specific tumor subtypes and to target less common tumors by pooling bio-specimen resources.
  • Collect data and create experiments.
    caB2B enables investigators to perform novel in silico experiments using already collected micro-array data sets. It helps in identifying up-regulated genes in specific cancers and also enable investigators to view data in the context of biological pathways. All of these lead to the more effective identification of novel drug targets to improves treatment strategies.
  • Perform various analysis and visualize analysis results by using various viewers
    caB2B enables individual users to work on common platforms to access caGrid resources from services in real time. Then visualize analysis results by using various viewers such as charts and dendrograms.
